# Splunk OpenTelemetry Collector Puppet Module

This is a Puppet module that will install and configure the Splunk
OpenTelemetry Collector.

[!WARNING] This Puppet Module is now being published in [Splunk's JFrog Artifactory](https://splunk.jfrog.io/ui/repos/tree/General/puppet-splunk),
and will no longer be published to Puppet Forge. Please refer to the [module's README](https://github.com/signalfx/splunk-otel-collector/blob/main/deployments/puppet/README.md)
for up to date information on how to install and utilize this module. [!WARNING]
